# Import session preparation

Before starting a replay, confirm:

1. **Target app exists** — created via `scripts/create.sh` and running
2. **Upstream repo URL** — the repo to import from
3. **Branch** — which upstream branch to replay (default: `main`)
4. **Starting point** — first commit (for fresh import) or where to resume

No session documents are needed — the upstream commit history is the plan.
